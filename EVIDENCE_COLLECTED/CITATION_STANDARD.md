# Evidence Citation Standard

**Date:** June 29, 2026
**Purpose:** Every claim must be directly traceable to a specific, verifiable source

---

## THE RULE

> **If you make a claim, you must provide a source. If you provide a source, it must be specific enough that a stranger can find and verify it.**

---

## CITATION FORMAT

### For Government Documents
```
Source: [Author/Agency], [Document Title], [Date] → [Section/Report Number]
Primary: [URL if available]
File: [Repository file path]
```

Example:
```
Source: Auditor General of Canada, Report 6, 2022 Fall Reports → "COVID-19 Vaccine Procurement and Distribution"
Primary: https://www.oag-bvg.gc.ca/internet/English/parl_oag_202210_06_e_43956.html
File: TOP_15_SUBSTANTIATION.md §1
```

### For Parliamentary Records
```
Source: House of Commons, [Record Type], [Date] → [Speaker/Topic]
Primary: https://www.ourcommons.ca/... or Hansard [Volume], [Page]
```

### For Court Documents
```
Source: [Court Name], [Case Name], [Case Number] → [Filing/Order]
Primary: [CanLII URL or Court PACER/CMECF link]
```

### For Corporate Filings
```
Source: [Company], [Filing Type], [Date] → [Section/Exhibit]
Primary: https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&CIK=[CIK]
```

### For News/Investigative Reports
```
Source: [Publication], "[Article Title]", [Author], [Date]
Primary: [URL or archive link]
```

### For Lobbying Registry
```
Source: Office of the Commissioner of Lobbying, Registry → [Registrant], [Communication Number], [Date]
Primary: https://lobbycanada.gc.ca/...
```

### For Corporate Registries
```
Source: [Jurisdiction] Corporate Registry → [Company Name], [Registration Number]
Primary: [Registry URL or search query]
```

### For ATIP Documents
```
Source: ATIP Request [Number], [Institution], [Date] → [Document Title/Description]
Status: [Released/Partially released/Denied]
```

---

## VERIFICATION CHECKLIST

Before any claim is marked complete, it must pass:

- [ ] **Specific document named** (not "sources say" or "reports indicate")
- [ ] **Page number, section, or paragraph referenced** where possible
- [ ] **Date of document provided**
- [ ] **Author/agency identified**
- [ ] **Primary source URL or repository file path included**
- [ ] **Quote or data point directly from the source** (not paraphrased from a secondary source)
- [ ] **Secondary sources flagged as such** (news reports citing documents are secondary; the document itself is primary)

---

## WHAT COUNTS AS PRIMARY

| Primary | Secondary |
|---------|-----------|
| Auditor General report | News article about Auditor General report |
| Parliamentary Hansard | Summary of parliamentary debate |
| SEC filing | Analysis of SEC filing |
| Court judgment | Commentary on court case |
| Lobbying registry entry | Article about lobbying |
| Corporate registry record | Profile about company |
| ATIP release | Report about ATIP findings |
| Government contract | News about contract award |

**Goal:** Every claim should cite primary sources. Secondary sources are acceptable only when primary is inaccessible, and must be flagged.
