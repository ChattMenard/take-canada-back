## WHAT IS THE LANDSCAPE?

The Landscape is a set of **four documents** designed to help you see the big picture before diving into the detailed evidence. They are maps, not the territory. Use them to orient yourself.

## HOW TO USE THE LANDSCAPE

### Scenario 1: "I know nothing. Where do I start?"

1. Read `LANDSCAPE_The_Ecosystem.md` from beginning to end (30–45 minutes)
2. Note which claims surprise you or seem most important
3. Use the Document Index at the end to find the detailed evidence file for those claims
4. Read the detailed evidence
5. Return to the Landscape to see how that detail fits the bigger picture

### Scenario 2: "Someone asked me to prove a specific claim."

1. Open `LANDSCAPE_Evidence_Verification_Matrix.md`
2. Find the claim by number or keyword
3. Check the verification level (✓✓✓, ✓✓, or ✓)
4. Read the Evidence File column to find the detailed substantiation
5. Cite the specific sources listed in the matrix

### Scenario 3: "Who is [person/company] and why do they matter?"

1. Open `LANDSCAPE_Who_is_Who.md`
2. Use Ctrl+F to search for the name
3. Read their profile, timeline, and connections
4. Follow the relationship maps to see their role in the system

### Scenario 4: "When did [event] happen, and what else was going on?"

1. Open `LANDSCAPE_Timeline_Master.md`
2. Find the date across all four tracks
3. Read the convergence moments to see how events in different domains intersected

### Scenario 5: "I want to educate someone else."

1. Give them `LANDSCAPE_The_Ecosystem.md` first
2. Let them read it and ask questions
3. Use the Verification Matrix to answer "how do you know?"
4. Use Who_is_Who to answer "who is that?"
5. Use the Timeline to answer "when did that happen?"

## PRINCIPLES

### 1. The Landscape Is a Map, Not the Territory

The Landscape simplifies for clarity. The Evidence documents contain the full citations, quotes, and primary sources. Always verify against the Evidence files before making public claims.

### 2. Verification Levels Matter

Use the ✓✓✓ claims when arguing with skeptics. Use the ✓ claims as leads for further research. Never present a ✓ claim as established fact.

### 3. The System Is the Story

Individual facts are less important than the pattern. The platform + the secrecy + the revolving door + the legal capture = a **system**. That is what the Landscape is designed to reveal.

### 4. This Is a Living Document

As new evidence emerges, update the matrices. As new cases are decided, update the legal precedents. As new leaks occur, update the secrecy claims.
